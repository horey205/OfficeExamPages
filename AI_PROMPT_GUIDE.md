# 🤖 AI 문제 출제용 프롬프트

아래 내용을 복사해서 NotebookLM이나 ChatGPT에 입력하세요.

## 1. 지적측량 (20문제 요청)

```text
이 문서를 바탕으로 지적측량 기출 변형 문제 20개를 만들어줘. 결과는 반드시 정해진 JSON 코드 형식이어야 해.

[
  {
    "num": "문 1.",
    "text": "문제 내용",
    "options": ["보기1", "보기2", "보기3", "보기4"],
    "answer": 1,
    "explanation": "해설 내용",
    "subject": "지적측량",
    "difficulty": "중"
  }
]
```
